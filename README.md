## Epiroc Ops Intel — one-page dashboard

Single-page, scroll-based operational intelligence simulation (black + yellow, glassmorphism, mining-tech UI).

### Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

### Notes

- The content uses **fictional but realistic** mining/ops metrics (KPI tiles, risk zones, workflow traces).
- If you want the “About” section to mirror your resume line-for-line, drop your resume file into this repo and tell me the filename—I’ll wire the exact metrics and wording into the modules.
